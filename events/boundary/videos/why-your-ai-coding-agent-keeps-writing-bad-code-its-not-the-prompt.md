# Why Your AI Coding Agent Keeps Writing Bad Code (It's Not the Prompt)

**Speaker:** Dex & Vaibhav
**Company:** Boundary / HumanLayer
**Channel:** Boundary
**Video:** https://www.youtube.com/watch?v=0LPBw3NO3Jc

## Summary

The episode argues that most AI coding failures start in the spec, not during implementation. It advocates splitting product design from technical design before code is written, using clearer specs and mockups to move cognitive burden upstream and make review easier later.

## Key Points

1. Better specifications lead to better implementations.
2. Split product design from technical design before coding starts.
3. Move cognitive burden upstream to reduce review pain later.
4. Mockups and rendered artifacts help surface issues earlier.
