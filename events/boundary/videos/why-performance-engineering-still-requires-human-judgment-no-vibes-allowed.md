# Why Performance Engineering Still Requires Human Judgment | No Vibes Allowed

**Speaker:** Dex & Vaibhav
**Company:** Boundary / HumanLayer
**Channel:** Boundary
**Video:** https://www.youtube.com/watch?v=mm6n4n09RaU

## Summary

This episode shows why performance optimization still depends on tight feedback loops and human judgment. The core message is that small mistakes can invalidate an optimization, so benchmark suites, profiling, and careful reading still matter even when AI helps with the work.

## Key Points

1. Performance engineering needs data-driven feedback loops.
2. Small mistakes can invalidate an entire optimization.
3. Benchmark suites and profiling should guide AI-assisted changes.
4. Humans still need to judge sensitive performance tradeoffs.
