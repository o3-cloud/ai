## Transcript Fetch Status

YouTube's API rate-limited our IP after the initial batch of 33 transcripts. The new batch of 25 talks could not be transcribed. To fetch transcripts for these talks:

1. Wait for the rate limit to reset (typically 24 hours)
2. Run the transcript fetcher script with the video IDs above
3. Or use `yt-dlp --write-auto-sub` to download subtitle files directly

```bash
# Example: fetch transcripts for new talks (after rate limit resets)
~/.venvs/catalog/bin/python /tmp/get_transcripts.py < /tmp/new_talks.json
```
