# Software Factories

Software factories represent the industrialization of AI-assisted
software development. This includes coding agents, spec-driven development,
and the infrastructure needed to build software at scale with AI.

Key themes from AI Engineer World's Fair 2026:

- **Spec-driven development** — Using specs to drive production workflows
- **Coding agents** — Building, deploying, and managing coding agents
- **Build systems** — Systems vs. code, and the future of software development
- **Agent skills** — Building great agent skills as the "missing manual"
