# Agent Memory & Continual Learning

Agent memory systems provide persistent context across sessions,
enabling agents to learn from past interactions and maintain long-term
knowledge. This includes note-based memory, graph-based memory, and
continual learning approaches.

Key themes from AI Engineer World's Fair 2026:

- **Note-based memory** — Turning notes into agent memory
- **A genius with amnesia** — The problem of context loss in agents
- **The log is the agent** — Using logs as agent memory
