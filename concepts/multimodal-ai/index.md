# Multimodal AI

Multimodal AI systems process and generate content across multiple
modalities — text, images, audio, and visual interfaces. This includes
vision models, voice interfaces, and browser-based agents.

Key themes from AI Engineer World's Fair 2026:

- **HTML for agent graphics** — Using HTML as the output format for agent-generated visuals
- **Voice in, visuals out** — Converting voice input to visual output
- **Browser agents** — Why browser agents need better vision, not better models
