# Agentic Engineering

Agentic engineering is the practice of building, deploying, and maintaining
AI agents that can autonomously perform tasks. It encompasses agent
architectures (single-agent, multi-agent, recursive), tool use, memory
management, and the infrastructure needed to run agents in production.

Key themes from AI Engineer World's Fair 2026:

- **Agent architectures** — Single-agent vs. multi-agent vs. recursive agent systems
- **Tool use & orchestration** — How agents select and use tools, the "100-tool trap"
- **Autonomous engineering orgs** — Agents that can build software autonomously
- **Production deployment** — Running agents at scale with reliability
