# Personal Agents & Claws

Personal agents are AI assistants that run on-device or locally,
providing personalized, private AI experiences. "Claws" refers to
personal AI tools that users can control directly.

Key themes from AI Engineer World's Fair 2026:

- **Local LLM terminals** — Building physical AI terminals for local agents
- **On-device agents** — Running agents locally for privacy and control
- **Personal agent UX** — Designing interactions for personal AI
