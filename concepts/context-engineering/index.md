# Context Engineering

Context engineering is the discipline of managing what information
enters an LLM's context window and how it's structured. It includes
retrieval-augmented generation (RAG), prompt engineering, context window
optimization, and cache strategies.

Key themes from AI Engineer World's Fair 2026:

- **RAG optimization** — Hybrid RAG, framework-free approaches, and raw SQL retrieval
- **Token efficiency** — Cutting tokens with local code indexes, JIT context
- **Context caching** — Extended Cache Augmented Generation (ECAG)
- **Retrieval boundaries** — How user signals die at retrieval boundaries
